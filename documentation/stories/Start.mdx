# Welcome to the KNIME Design System

The **KNIME Design System (KDS)** is a Vue 3 component library providing reusable, accessible, and consistent UI components for KNIME applications.

## What's included

Browse the sidebar to explore the following categories:

### Core Styles

- **Icons** — Browse all available icons and compare source vs. optimized versions.
- **Design Tokens** — Explore colors, typography, spacing, and other CSS custom properties.

### Components

- **Accessories** — Visual helper elements such as icons, spinners, and avatars.
- **Buttons** — Clickable actions including standard buttons, link buttons, and toggle variants.
- **Form Fields** — Data entry components like text inputs, checkboxes, and radio buttons.
- **Overlays** — Floating layers such as modal dialogs and popovers.
- **Structures** — Content containers like cards and empty state placeholders.
